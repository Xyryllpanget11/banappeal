Privacy Policy

This Privacy Policy applies to the use of the Discord Ban Appeals program and Discord Ban Appeals bot (referred to as "DBA"). The following outlines the terms and conditions of the use of DBA, which applies to all users.

DEFINITIONS

DBA refers to the Discord Ban Appeals program and/or the Discord Ban Appeals discord bot. "You" refers to the User. "Us and We" refers to the Discord Ban Appeal Team. "Bot" refers to the Discord Ban Appeal Bot.

INFORMATION SHARING

DBA (Discord Ban Appeals) bot will not share any information unless requested or needed for support reasons. DBA will not knowingly take any personal information. If personal information is inadvertently collected, it will be deleted as soon as possible to prevent any harm to the user.

Discord Ban Appeals as a whole will not share any information without the permission of a Discord Server or targeted member. If you have questions about our policy, you can join our support server.

DATA COLLECTION

DBA bot records the following information:

  • Command Run
  • Time the command was used
  • What user was banned
  • Why was the user banned
  • What server was the user banned from
  • If the bot is DMed
  • What User interacted with the bot
  • The server the command or interaction occurred
  • If the bot is added to a server
  • What server the bot was added to
  • If the bot is removed from a server
  • What server the bot was removed from

This information is collected to improve the bot's functionality and identify any potential violations of Discord's Terms of Service.

DATA DELETION

If you, as the user, would like specific information to be deleted, you may join our support server and request that the information be deleted. We will delete any information we have about a specified topic or event upon request, provided that you have the proper authority and the event involves you, your server, or your targeted member.

DBA does not intentionally collect personal information. However, if personal information is inadvertently collected for any reason, you may contact us to delete the information if it has not already been cleared. We will not share any personal information we find in our logs with anyone except the server from which the user was banned, DBA staff, and the banned user.

MODIFICATIONS

By using DBA, you agree to the terms and conditions of this Privacy Policy. We reserve the right to modify this policy at any time, and any changes will be effective immediately upon posting the updated policy on our website.
