(DBA) Discord Ban appeals

DBA aka Discord Ban Appeals bot is a bot that is supposed to help your server with the ban appeal process!

DBA has features that will hopefully help banned users appeal bans, so you don't need to manually DM everyone the link!

  • - DBA has a /appeal command for manually requesting the ban appeal link
  • - DBA has a /about command so users can get info about the bot instantly
  • - All slash commands are EMPHERICAL and will not be shown to other members, meaning it causes no clutter and other members don't know about you using the command.
  • - The bot will respond with information upon being DMed, this way you can get the link in the offical server or servers where text channels are not avalible, EG: if you are muted and appealing a mute, DM the bot for the link
  • - *DBA will DM users that get banned from a server that the bot is also in

* The bot will not DM users if the bot is not in at least 2 servers the bot is also in! Make sure to have members join the ban-appeals support discord to prevent this error
